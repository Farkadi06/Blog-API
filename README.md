## What this App does?

- By this app, you will be able to:

  - [x] Create a new Blog.
  - [x] Edit a Blog.
  - [x] Delete a Blog.
  - [x] Get list of all Blogs.
  - [x] Get list of blog by Collection.
  - [x] Get list of Blogs by Author.
  - [x] Create a new User(Author).
  - [ ] Edit your user profile info.
